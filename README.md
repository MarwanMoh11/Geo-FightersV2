**Geofighters 2.0** is a high-performance, fast-paced 3D arcade shooter built with **Three.js**, **Vite**, and **Miniplex (ECS)**. Engage in geometric warfare with fluid controls and intense combat.

## ✨ Features

- 🎮 **Fluid Controls**: Responsive movement and aiming systems.
- 🧊 **3D Graphics**: Modern geometric aesthetics powered by Three.js.
- ⚡ **ECS Architecture**: Highly optimized performance using Miniplex.
- 🛠️ **Modular Design**: Clean separation of concerns with a robust system-based logic.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/marwan/geofighters-2.0.git
   ```

2. Navigate to the project directory:
   ```bash
   cd geofighters-2.0
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the project for production:
```bash
npm run build
```

## 🏗️ Project Structure

- `src/core/`: Essential game loop, world, and renderer setup.
- `src/systems/`: ECS systems handling game logic (Input, Physics, Render, etc.).
- `src/entities/`: Entity definitions and components.
- `src/assets/`: 3D models, textures, and sounds.
