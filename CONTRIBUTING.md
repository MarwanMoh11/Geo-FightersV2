# Contributing to Geofighters 2.0

First off, thank you for considering contributing to Geofighters 2.0! It's people like you that make this project better.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and provide steps to reproduce it.
- Include details about your environment (OS, Browser version).

### Suggesting Enhancements
- Open a GitHub Issue with the "enhancement" label.
- Explain why this feature would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Install dependencies: `npm install`.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes.
5. Make sure your code follows the existing style (run `npm run lint`).
6. Issue that pull request!

## Code Style
Please follow the standard TypeScript and Three.js practices. We use ESLint and Prettier to maintain code quality.
